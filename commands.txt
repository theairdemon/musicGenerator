Git Commands

git status = check the status of unadded/uncommitted files
git add . = add all unadded files to commit queue
git commit -am "msg" = commit all files in queue with a msg attached
git branch = check the branch you're on
git branch -m master [name] = swap to name of the other branch from master
git push -u origin [name] = push all commits to the branch, will be published on GitHub
